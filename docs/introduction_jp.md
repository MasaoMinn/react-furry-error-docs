# react-furry-error

可愛くて実用的な React 開発期エラーオーバーレイ 🐾  
頭を悩ませるエラーを、もふもふのキャラクターのリアクションに変えます。

---

`react-furry-error` は **React プロジェクト向けの開発期エラー表示ツール** です。

開発中のアプリケーションでエラーが発生した場合、冷たい赤いエラーページだけを表示するのではなく、エラーの種類に応じて **さまざまな表情や動作の React キャラクターイラスト** を表示し、問題の大まかな方向をすぐに理解できるように助けます。

もちろん、可愛さだけではなく情報が不足することはありません ——  
**元のエラー情報（message、stack、ファイル位置）はすべて完全に保持され、表示されます**。

### このプロジェクトの特徴は？

- 🐾 エラーを「視覚的に分類」
- 🔍 元のエラー情報を一切隠しません
- ⚙️ React の内部実装に侵入しません
- 🧩 Vite / Webpack / CRA / Next.js（ランタイムエラー）で動作可能

`react-furry-error` の目標はエラーを隠すことではなく、  
**エラーをより速く理解できるようにすること** です。
