# react-furry-error

一个可爱且实用的 React 开发期错误覆盖层 🐾
让令人头疼的报错，变成毛茸茸的兽设反应。

---

`react-furry-error` 是一个 **用于 React 项目的开发期错误展示工具**。

当你的应用在开发过程中发生错误时，它不会只显示冷冰冰的红色报错页面，而是会根据错误类型，展示 **不同表情和动作的 React 兽设插画**，帮助你在第一时间理解问题的大致方向。

当然，可爱并不意味着信息缺失 ——  
所有 **原始错误信息（message、stack、文件位置）都会被完整保留并展示**。

### 这个项目有什么不同？

- 🐾 将错误「可视化分类」
- 🔍 不吞掉任何原始错误信息
- ⚙️ 不侵入 React 内部实现
- 🧩 可运行于 Vite / Webpack / CRA / Next.js（运行时错误）

`react-furry-error` 的目标不是隐藏错误，  
而是 **让你更快地理解错误**。
